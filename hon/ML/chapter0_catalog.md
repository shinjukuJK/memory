># 机器学习Machine Learning概览

>## Supervised Learning
>### 通用
>* x→f(x)→y，最普通的x和y都存在

>## Self-supervised Learning
>### 通用
>* x→f(x)→y，适用于x不充足的情况下

>## Generative Adversarial Network
>### 通用
>* x→f(x)→y，适用于有x集，有y集，但无法一一对应

>## Reinforcement Learning
>### 通用
>* x→fx→y，已知x未知y，但可以给出y的好坏评价

>## Anomaly Detection
>### 通用
>* x→f(x)→y，若y为classification，若x不存在于现有class内，给出异常警告

>## Explainable AI
>### 通用
>* 不仅能得出y，还能给出fx

>## Model Attack
>### 通用
>* 针对x的特殊处理，可能成为对f的攻击
